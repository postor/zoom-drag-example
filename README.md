# 缩放拖拽示例

接近原生，紧使用了jQuery，方便理解又比原生省代码，主要是供对手势操作细节有定制需求的同学参考

## 启动

```
yarn # 安装包
yarn dev # 启动开发环境
```

默认地址： http://localhost:1234 具体参考控制台输出
